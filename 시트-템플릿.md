# 구글 시트 템플릿 구조

구글 시트에 아래 4개의 시트(탭)를 만드세요.
각 시트의 **첫 번째 행**은 헤더(열 이름)입니다.

---

## 시트 1 — 기본정보

A열(항목) / B열(내용)

| 항목 | 내용 |
|------|------|
| clinicName | 미용한의원 |
| tagline | 건강한 아름다움을 위한\n한의학적 솔루션 |
| address | 서울시 강남구 논현동 000-0 |
| phone | 02-000-0000 |
| hours | 월~금 10:00-19:00  /  토 10:00-15:00 |

> `\n` 을 쓰면 홈페이지에서 줄바꿈됩니다.

---

## 시트 2 — 이달의이벤트

헤더: 이벤트명 / 원가 / 이벤트가격 / 기간 / 설명 / 표시

| 이벤트명 | 원가 | 이벤트가격 | 기간 | 설명 | 표시 |
|---------|------|-----------|------|------|------|
| 피부 재생 침 패키지 | 300000 | 180000 | 2026년 6월 | 한방 피부 재생 침 패키지 | 예 |
| 체형 교정 케어 3회 | 500000 | 320000 | 2026년 6월 | 체형 교정 집중 케어 | 예 |

> **표시** 열: `예` → 보임 / `아니오` → 숨김  
> **원가·이벤트가격**: 숫자만 입력 (콤마 없이)  
> 할인율(% OFF)은 자동 계산됩니다.

---

## 시트 3 — 일반프로그램

헤더: 카테고리 / 프로그램명 / 가격 / 설명 / 표시

| 카테고리 | 프로그램명 | 가격 | 설명 | 표시 |
|---------|-----------|------|------|------|
| 피부 치료 | 기본 침술 | 50000 | 경혈 자극으로 기혈 순환 개선 | 예 |
| 피부 치료 | 화침 시술 | 80000 | 피부 재생 및 흉터 개선 | 예 |
| 체형 교정 | 골반 교정 | 100000 | 골반 불균형 정렬 치료 | 예 |
| 한방 다이어트 | 다이어트 침 | 70000 | 식욕 억제 & 대사 증진 | 예 |

> **카테고리**가 같은 항목끼리 자동으로 묶여서 표시됩니다.  
> 가격에 "상담 후 결정" 처럼 글자를 입력해도 됩니다.

---

## 시트 4 — 후기사진

헤더: 이미지URL / 캡션 / 표시

| 이미지URL | 캡션 | 표시 |
|----------|------|------|
| https://... | 피부 재생 후기 | 예 |
| https://... | 체형 교정 후기 | 예 |

> **이미지 URL 얻는 방법 (구글 드라이브)**  
> 1. 구글 드라이브에 사진 업로드  
> 2. 사진 우클릭 → [공유] → "링크가 있는 모든 사용자"로 변경  
> 3. 링크 복사: `https://drive.google.com/file/d/파일ID/view`  
> 4. 아래 형식으로 변환해서 시트에 입력:  
>    `https://drive.google.com/uc?id=파일ID`

# 전체 설정 순서

1. **구글 시트 만들기**  
   - 위 4개 시트(탭) 생성 및 데이터 입력

2. **Apps Script 설정**  
   - 시트 상단 메뉴 → [확장 프로그램] → [Apps Script]  
   - `apps-script.js` 파일 내용 전체 복사 → 붙여넣기 → 저장

3. **배포**  
   - [배포] → [새 배포] 클릭  
   - 유형: **웹 앱**  
   - 실행: **나 (본인)**  
   - 액세스: **모든 사용자**  
   - [배포] 버튼 클릭 → URL 복사

4. **index.html 수정**  
   - `index.html` 파일을 텍스트 편집기로 열기  
   - `YOUR_APPS_SCRIPT_URL_HERE` 부분을 복사한 URL로 교체

5. **Vercel 에 올리기 (무료)**  
   - https://vercel.com 접속 → 구글 계정으로 가입  
   - [Add New Project] → [Upload] 선택  
   - `미용한의원` 폴더를 드래그 앤 드롭  
   - 배포 완료 → 링크 생성!

# 이후 수정 방법

**내용 바꾸고 싶을 때** → 구글 시트 열어서 수정  
→ 홈페이지 새로고침하면 바로 반영됩니다.

**사진 바꾸고 싶을 때** → 구글 드라이브에 사진 업로드  
→ URL 변환 후 후기사진 시트에 붙여넣기

추가 수정이 필요하면 언제든지 Claude Code에 말씀해 주세요!
